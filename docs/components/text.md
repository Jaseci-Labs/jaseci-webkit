#### Summary

A `Text` component is a that renders elements in various typographic-styles.

<u>Example</u>

```JSON
{
	"component": "Text",
	"props": {
		"variant": "simple"
	}
}
```

#### Props

- variant - sets the text style
  - Acceptable values are - `simple` | `title`
